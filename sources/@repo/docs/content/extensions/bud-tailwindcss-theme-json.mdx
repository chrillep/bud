**@roots/bud-tailwindcss-theme-json** is a **bud.js** extension used to utilize values from `tailwind.config.js` in your `theme.json` file.
It requires [@roots/bud-wordpress-theme-json](/extensions/bud-wordpress-theme-json) to be installed as a peer dependency.

For background information check the [WordPress theme.json documentation](https://developer.wordpress.org/block-editor/how-to-guides/themes/theme-json/).
